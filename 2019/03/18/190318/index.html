丢番图方程

定义：含有两个未知量𝑥, 𝑦且形如𝑎𝑥 + 𝑏𝑦 = 𝑐的方程。

我们本次要讨论的问题:

  1. 寻找方程的一个解
  2. 寻找方程的所有解（通解）
  3. 寻找方程的在某区间内的解及解的个数
  4. 寻找方程的解中 x+y 的最小值

我们将忽略𝑎 = 𝑏 = 0的情形，因为这种情况下不是无穷多解就是无解。（取决于𝑐的取值）
